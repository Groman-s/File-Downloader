# Файловый хостинг

Это простая утилита на Go для создания ссылок на скачивание файлов.

## Подготовка

Запустите файл впервые. Создастся конфиг по умолчанию со следующим содержимым:
```
port: 8085

pages:
- page: /example
  path: files/example.txt
```
Остановите сервис и настройте его. Укажите страницы и файлы, к которым они ведут. Например, в стандартном конфиге при обращении к адресу `localhost:8085/example` будет скачан файл `example.txt` из директории `files`, расположенной в директории сервиса.

## Дополнительная настройка

Для предоставления доступа к скачиванию файлов с вашей системы извне необходимо открыть указанный в конфиге порт. Сделать это можно, например, через утилиту ufw. Для стандартного порта 8085 нужно прописать команду `sudo ufw allow 8085/tcp`.
